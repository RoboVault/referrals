<!-- GETTING STARTED -->
## Getting Started

### Install Dependencies 

Requires npm and pip. See .github/workflows for suitable versions. 

```sh
npm install
pip install requirements-dev.txt
```

### Build

```sh
brownie compile
```

### test

```sh
brownie test
```

### lint

```sh
npm run lint
```
